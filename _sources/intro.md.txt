# Installation using ```pip```
Clone the repository to a local folder using
```bash
git clone https://github.com/jmiramont/mcsm-benchmarks.git
```
Then install the toolbox using:
```bash
pip install mcsm-benchmarks/dist/mcsm_benchmarks-0.1.0-py3-none-any.whl
```

# Installation using ```poetry```

If you use [```poetry```](https://python-poetry.org/docs/) for management dependency and packaging in Python, you can add the toolbox to your dependencies using:

```bash
poetry add git+https://github.com/jmiramont/mcsm-benchmarks.git
```
